# Поведение
Делится на Behaviour и Activity

Behaviour - основной класс поведения, такие как движение, атака и тд
Activity - класс для внутренней работы управляется через Behaviour

Behaviour бывает двух типов:

Behaviour - поведения чаще всего вызываемые из вне при надобности
BehaviourNode - поведение работающее внутри древа поведений
