🛡️ Tower Defense (Unity)

Модульная Tower Defense игра, построенная с упором на расширяемость, переиспользуемость и чистую архитектуру.

🧩 О проекте

Основная идея проекта — создать систему, в которой добавление новых механик, эффектов и поведения не требует изменения существующего кода, а достигается за счёт композиции и слабой связанности компонентов.

Проект ориентирован на демонстрацию архитектурных решений и подходов к разработке, применимых в production-среде.

⚙️ Архитектура
Компонентный подход

Игровая логика разделена на независимые компоненты, которые можно свободно комбинировать между собой.
Это позволяет:

легко добавлять новые типы башен и врагов
переиспользовать существующие элементы
избегать жёсткой связанности
Event Bus (шина событий)

Взаимодействие между системами построено через событийную шину.

Это даёт возможность:

внедрять новые системы без изменения core-логики
уменьшать зависимости между модулями
гибко расширять функциональность проекта
Behaviour Tree (самописный)

Для управления поведением врагов используется собственная реализация behaviour tree.

Особенности:

гибкая настройка логики поведения
возможность легко добавлять новые ноды
разделение логики принятия решений и исполнения
🎯 Ключевая цель

Сделать систему, в которой:

добавление новых эффектов, способностей и поведения происходит быстро и без изменения существующего кода

🚀 Возможности
модульная система эффектов
расширяемое поведение врагов
гибкая система взаимодействия между системами
готовность к масштабированию
🛠 Стек
Unity (C#)
Component-based architecture

Custom Behaviour Tree
📌 Примечание

Проект создан с акцентом на архитектуру и демонстрацию инженерных подходов, а не только на реализацию геймплея.
